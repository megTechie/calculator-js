# 🧮 Calculator-JS

A simple and responsive calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations through an intuitive user interface and demonstrates fundamental JavaScript concepts such as DOM manipulation and event handling.

## ✨ Features

* Basic arithmetic operations (+, −, ×, ÷)
* Clear display functionality
* Responsive user interface
* Built with vanilla JavaScript

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## 📁 Project Structure

```text
calculator-js/
├── index.html
├── style.css
├── tool.js
└── README.md
```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/megTechie/calculator-js.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.
